# Delivery Arretado

1. Install composer

      ```text
      $ cd deliveryarretado
      $ composer install
      ```

2. Run php server

      ```text
      $ cd deliveryarretado
      $ php -S 127.0.0.1:8888 -t web web/index.php
      ```

3. Open browser

      Open http://127.0.0.1:8888 in your browser
